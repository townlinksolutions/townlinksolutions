## ğŸ”§ Customization

### Adding Images

To add your beach images to the website:

1. **Hero Background Image**:
```javascript
// In script.js, uncomment and update:
setHeroBackground('path/to/your/hero-image.jpg');
```

2. **Gallery Images**:
```javascript
// In script.js, uncomment and update:
loadGalleryImage('path/to/your/gallery-image.jpg');
```

Or directly replace the `.gallery-placeholder` divs in `index.html` with:
```html
<img src="path/to/image.jpg" alt="Parna Kuteera" style="width: 100%; height: 100%; object-fit: cover;">
```
