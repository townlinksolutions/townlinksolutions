# Masthappa Naik Balase - Personal Branding Website

A professional political branding website for Masthappa Naik Balase, a prominent social activist, businessman, and environmental advocate from Murdeshwar and Bhatkal regions of Uttara Kannada district, Karnataka.

## 🌟 Features

### Public Pages (Kannada Content)
- **Home**: Hero section, quick stats, timeline preview, focus areas, recent news
- **About**: Detailed biography, early life, business journey, philosophy
- **Vision**: Focus areas and priorities for Uttara Kannada development
- **Achievements**: Awards, recognitions, and milestones
- **Timeline**: Complete chronological journey from 2000-2026
- **News**: Latest updates and media coverage
- **Events**: Upcoming and past events
- **Gallery**: Photo gallery of social work and campaigns
- **Community**: Get involved section
- **Contact**: Contact form and information

### Admin Features (Demo Mode)
- **Login System**: Demo credentials - username: `admin`, password: `admin123`
- **Dashboard Analytics**:
  - Total visitors, messages, locations, time on site
  - Visitor trends chart
  - Major concerns analysis (Health, Environment, Education, Jobs, Infrastructure)
  - Age demographics
  - Geographic distribution
  - Device usage stats
  - Popular pages
  
- **Feedback Management**:
  - View all messages/feedback
  - Filter by: All, Unread, Important
  - Mark as read/unread
  - Archive messages
  - Reply functionality (demo)
  
- **Content Update**:
  - Add news articles
  - Add events
  - Upload gallery images
  - Update homepage banner
  - (All updates are demo mode - not actually saved)

## 🚀 Getting Started

### Option 1: Open Directly
Simply open `index.html` in any modern web browser.

### Option 2: Local Server (Recommended)
For better experience, use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using PHP
php -S localhost:8000

# Using Node.js (http-server)
npx http-server
```

Then open `http://localhost:8000` in your browser.
