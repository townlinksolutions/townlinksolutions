# Getting Started Guide
## Masthappa Naik Balase - Political Branding Website

### 🚀 Quick Start (Step by Step)

#### Step 1: Download/Extract Files
Extract all files to a folder on your computer.

#### Step 2: Open the Website
**Option A - Direct Open (Simplest)**
- Navigate to the folder
- Double-click on `index.html`
- The website will open in your default browser

**Option B - Local Server (Recommended for Full Features)**
```bash
# If you have Python installed:
python -m http.server 8000
# Then open: http://localhost:8000

# If you have PHP installed:
php -S localhost:8000
# Then open: http://localhost:8000

# If you have Node.js installed:
npx http-server
# Then open the URL shown in terminal
```
